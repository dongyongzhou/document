##1 Overview

###1.1 术语

- FAT：File Allocation Table (文件分配表)
- MBR：Master Boot Record (主引导记录)
- DBR：DOS Boot Record (DOS引导记录，位于分区引导扇区)
- DPT：Disk Partition Table (磁盘分区表)
- BPB：BIOS Parameter Block (BIOS 参数块)
- Sector：扇区
- Cluster：簇
